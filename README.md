# MentorNet AI: Elite Academic Networking Platform

MentorNet is a state-of-the-art, AI-driven academic networking platform designed to connect students with mentors through semantic intelligence, real-time collaboration, and multi-dimensional impact analytics.

## 🚀 13-Phase Production Engineering

This project was built through a systematic 13-phase engineering process, evolving from a core vector-search prototype to a hardened enterprise ecosystem.

1.  **AI Semantic Core**: Integrated FAISS vector store with OpenAI-compatible embeddings for sub-second researcher matching.
2.  **Multidimensional Ranking**: Developed a ranking engine that balances research overlap, expertise, and institutional affinity.
3.  **Real-Time Infrastructure**: Built a persistent Socket.io communication layer for instant peer-to-peer messaging.
4.  **Mentorship Scheduling**: Implemented a conflict-aware booking system with automated availability management.
5.  **Intelligence Analytics**: Deployed AI-driven sentiment analysis to quantify the "vibe" and quality of mentorship interactions.
6.  **Redis-Backed Caching**: Optimized dashboard latency with high-performance caching for personalized feeds.
7.  **Enterprise Multi-Tenancy**: Engineered an organizational layer for institutional university deployments.
8.  **Security Hardening**: Implemented Redis-based rate limiting and comprehensive audit logging for compliance.
9.  **AI Writing Assistant**: Integrated an LLM-based assistant to help students draft personalized outreach messages.
10. **Gamification & Impact**: Created a global leaderboard based on a proprietary contribution scoring algorithm.
11. **Production Orchestration**: Dockerized the entire stack (PostgreSQL, Redis, FastAPI, Next.js) for cloud readiness.
12. **Premium UX/UI**: Designed a high-fidelity dashboard with glassmorphism, micro-animations, and real-time toasts.
13. **Explainable AI (XAI)**: Implemented a transparency engine that explains *why* a mentor was matched to a student.

## 🛠️ Technology Stack

- **Backend**: FastAPI (Python 3.11), SQLAlchemy, PostgreSQL
- **Frontend**: Next.js 14, TailwindCSS, Framer Motion
- **AI/ML**: FAISS (Facebook AI Similarity Search), Sentence Transformers, Custom Sentiment Heuristics
- **Infrastructure**: Docker, Redis (Caching & Rate Limiting), Socket.io (Real-time)
- **Security**: JWT Auth, Institutional Audit Logging, Client-side Rate Limiting
